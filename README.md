# goshxt  
  
How to use:

create a json file `config.json` under src  
![alt text](https://github.com/594-666/goshxt/blob/main/demo.png?raw=true)  
in config.json, add:  
```
{
    "student_id": "",
    "password": ""
}
```
and fill in your id & password  
then open a terminal in the folder of this repo  
run `npm start`
